Vulkan game engine in C

A game engine programmed in C using Vulkan as a didactic exercise to learn game engine development at its lowest level.

Repository
C vulkan game engine development

Description

This is a 3D game engine made in C with Vulkan (with plans to support DirectX and OpenGL) from scratch. The project started in September 2024, after finishing a first iteration of ModelOne, because of the passion and the willingness to learn about game engine development. At the moment this engine is in a very early stage.
