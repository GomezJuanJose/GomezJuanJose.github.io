Operation Crimson Day

Top down shooter with a wave system, entity manager system and dynamic music system when the difficulty increases.

Itch.io
cpp unreal-engine game-development

Description

Top down shooter on 2.5D pixel art where there are objects with a pillar form which attracts strange creatures. Your mission is kill as much as you can and delay the invasion.

Features

  • Wave system.
  • UI system.
  • Dynamic system
  • Ranking system.
  • Compatible with controller.
  • Four levels with different difficulties.
  • Six different guns.
  • Different types of enemies.
  • Translated to english, spanish and japanese.
