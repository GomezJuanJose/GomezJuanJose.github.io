Una Aventura Insesperada

An short puzzle videogame for NDS where the player must go throught a dungeon in a limited number of movements.

Itch.io

Description

An short puzzle videogame for NDS where the player must go throught a dungeon in a limited number of movements, once it reach the end an entity will ask some questions, answer right or you will start again.

Note

If the inputs not works on your emulator try to use No$GBA.

Features

  • Pass throught five levels in a limited number of movements.
  • Respond correctly a quiz after each level.
